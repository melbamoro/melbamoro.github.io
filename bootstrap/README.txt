How to Update Bootstrap Framework?

IDea is a static HTML template. You need to update the Bootstrap
framework manually. We highly recommend you check or watch the repository
https://github.com/twbs/bootstrap/releases of the Bootstrap framework for
new releases, and update it on your website if there are new releases.

IDea Responsive Website template is compatible with Bootstrap 3.x version.

In order to update the bootstrap framework you need to download the
latest 3.x version from its repository https://github.com/twbs/bootstrap
and replace the files of html/bootstrap folder in the template,
with the updated files.
